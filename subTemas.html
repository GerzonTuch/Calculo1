Se presentan los subtemas de timites

• Método Gráfico
El método gráfico para encontrar límites implica el uso de una gráfica de la función para observar el comportamiento de la función a medida que la variable se acerca a un punto específico. Al analizar la gráfica, se puede estimar el valor al que la función parece estar aproximándose.

Este método es útil para obtener una comprensión visual del límite. Al trazar la gráfica de f(x), se observa el valor de f(x) cuando x se acerca al punto de interés a desde ambos lados (izquierda y derecha). La observación visual puede ayudar a identificar el límite, especialmente en casos de discontinuidades o comportamientos oscilatorios.
• Método Numérico
El método numérico para calcular límites consiste en aproximar el valor del límite utilizando una tabla de valores de la función cerca del punto de interés. Se evalúa la función en varios puntos que se acercan cada vez más al valor de x en el que se desea encontrar el límite.

Se crea una tabla con valores de x que se acercan a a tanto por la izquierda como por la derecha. Los correspondientes valores de f(x) se calculan y se examinan para ver si convergen a un número específico. • Método Analítico
El método analítico para encontrar límites utiliza técnicas algebraicas y propiedades de las funciones para calcular el límite de manera exacta. Este método implica la manipulación de la expresión matemática de la función para encontrar el valor al que la función se aproxima.

Este enfoque incluye la simplificación de la función, la factorización, la racionalización y la aplicación de propiedades de los límites. Para funciones continuas, el límite se puede encontrar simplemente evaluando la función en el punto de interés. Para funciones más complejas, se utilizan teoremas y técnicas avanzadas como el Teorema del Sándwich, límites laterales, y el uso de límites especiales. Este método es generalmente más preciso que los métodos gráfico y numérico, ya que proporciona un valor exacto del límite.
